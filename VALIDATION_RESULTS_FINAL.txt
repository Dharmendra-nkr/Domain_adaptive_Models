## ğŸ”¬ Validation Methodology

### Data Processing:
1. Loaded BC5CDR test set (500 documents)
2. Tokenized with BioBERT tokenizer
3. Aligned labels at word-piece level
4. Created evaluation batches (batch size 32)

3. Predicted token labels
